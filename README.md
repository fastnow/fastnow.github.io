# FastNow Studio - 创新网页工具与应用集合

[![GitHub](https://img.shields.io/badge/GitHub-fastnow-blue?logo=github)](https://github.com/fastnow)
[![License](https://img.shields.io/badge/License-BSD_3--Clause-blue)](LICENSE)
[![Web Tools](https://img.shields.io/badge/%E7%BD%91%E9%A1%B5%E5%B7%A5%E5%85%B7-20%2B-brightgreen)](https://fastnow.github.io)

FastNow Studio 是一个专注于开发创新网页工具与应用的平台，所有项目均为纯前端实现，无需安装即可在浏览器中直接使用。

## 🌟 特色功能

### 🛠️ 实用工具类
- **FN Python在线编辑器** - 在浏览器中安全运行Python代码
- **面部比例评定系统** - 科学分析面部比例并提供美容建议
- **秘加块·伪诗词加密系统** - 将文本转化为意境诗词进行加密
- **HTML/CSS/JS在线编辑器** - 实时编写和预览网页代码
- **十年·PCR文献资料库** - 重要历史文献在线阅读

### 🎮 游戏/模拟类
- **五子棋AI** - 挑战人工智能五子棋对手
- **2D星体运行模拟** - 探索宇宙星体运动规律
- **康威的生命游戏** - 经典细胞自动机模拟

### 🎵 媒体娱乐类
- **音乐聚合解析** - 播放多平台优质音乐
- **在线FM播放器** - 收听各类电台节目
- **pixiv随机ST展示器** - 随机展示优质插画

### 🌐 网页应用类
- **DashSaver屏保仪表盘** - 精美的纯前端屏幕保护
- **毛泽东选集在线阅读** - 完整在线阅读体验
- **随机二次元图片API** - 为开发者提供图片接口服务

## 📦 技术特性

- **纯前端实现**：所有工具均在浏览器中运行，无需后端服务器
- **响应式设计**：适配各种设备和屏幕尺寸
- **暗色/亮色主题**：支持一键切换主题模式
- **实时搜索**：快速查找所需工具
- **项目分类**：按用途分类管理，方便查找

## 🚀 快速开始

1. 访问 [FastNow Studio 主页](https://fastnow.github.io)
2. 浏览项目列表或使用搜索功能
3. 点击任意项目卡片即可使用对应工具
4. 推荐收藏页面以便下次快速访问
