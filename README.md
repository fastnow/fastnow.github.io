# FastNow Studio - 创新网页工具与应用集合

[![GitHub](https://img.shields.io/badge/GitHub-fastnow-blue?logo=github)](https://github.com/fastnow)
[![License](https://img.shields.io/badge/License-BSD_3--Clause-blue)](LICENSE)

在校学生独立维护的项目集合。所有工具均为纯前端实现，开箱即用。

## 项目列表

### 实用工具
- **FSChart** - 自然语言生成 2D/3D 图形，集成 Pollinations AI
- **FN Python在线编辑器** - 浏览器内安全运行 Python 代码
- **JS混淆加密工具** - 十余项 JavaScript 代码混淆功能
- **在线排序算法可视化** - 动态演示常见排序算法
- **Base64全能转换** - 支持文本/文件/图片/PDF/JSON 编解码
- **面部比例评定** - 基于三庭五眼的美容建议
- **秘加块·伪诗词加密** - 文本转化成意境诗词加密
- **小天才校验码** - 小天才 v2~v3 校验码计算
- **网站源代码获取** - 在线查看网页 HTML 源码
- **DashSaver屏保仪表盘** - 时间/名言/搜索/快捷链接
- **颜文字字典** - 分类检索，一键复制

### 游戏模拟
- **FastCraftX** - 浏览器直玩 Minecraft 1.8
- **F(AST)＋Launcher** - Minecraft JAVA 启动器
- **2D星体运行模拟** - 宇宙星体运动模拟
- **五子棋AI** - 挑战人工智能对手
- **康威的生命游戏** - 经典细胞自动机

### 媒体娱乐
- **音乐聚合解析** - 播放网易云/抖音/5sing 等平台音乐
- **在线FM播放器** - 收听各类电台节目
- **pixiv随机展示器** - 随机浏览 pixiv 插画

### 网页应用
- **十年·PCR文献资料库** - 重要文献在线阅读
- **毛泽东选集在线阅读** - 章节导航与搜索
- **HTML/CSS/JS编辑器** - 在线编码与实时预览
- **随机二次元图片API** - 开发者图片接口
